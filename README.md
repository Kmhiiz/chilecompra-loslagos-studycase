# 📊 Auditoría de Gasto Público Regional
## ChileCompra — Modalidad Trato Directo | Región de Los Lagos (2023-2025)

Pipeline analítico end-to-end para auditar el gasto público bajo 
modalidad Trato Directo en la Región de Los Lagos, Chile.

## 🛠️ Stack Técnico
- **SQL** (SQLite / DBeaver) — ETL y unificación de 12 datasets
- **Python** (Pandas) — Análisis exploratorio y motor de alertas
- **Notion** — Documentación y storytelling

## 🔍 Hallazgos Principales
- Cuello de botella operacional: 34% de OC estancadas en estado "Aceptada"
- Micro-monopolios encubiertos bajo mercado aparentemente diversificado
- Servicio de Salud Osorno: +$20.5M sobre el promedio regional por OC
- Caso Cochamó: anomalía detectada mediante análisis exploratorio humano
